# Bhasha-Kahani
An multilingual storytelling platform

Bhasha Kahani is a modern multilingual storytelling platform where users can create, share, and discover stories in different languages while connecting with a community of readers and writers. The platform supports various story genres, including folk tales and mythology, allowing users to preserve cultural narratives and express their creativity through engaging written content.

The application offers a seamless social storytelling experience with features such as user authentication, story publishing, likes, follow/unfollow functionality, personalized profiles, and multilingual support. Designed with a clean dark-themed interface, responsive layout, and intuitive navigation, Bhasha Kahani provides an enjoyable reading and writing experience across devices while promoting regional languages, cultural heritage, and collaborative storytelling.

Key Features
🔐 Secure user authentication (Sign Up & Login)
📖 Create, publish, and manage stories
🌍 Multilingual support (English, Telugu, Hindi)
🏷️ Story categories such as Folk Tale and Mythology
❤️ Like stories and engage with the community
👥 Follow and unfollow storytellers
👤 Personalized user profiles with followers, following, story count, and total likes
📱 Fully responsive and modern dark-themed user interface
🎨 Clean card-based design for an enhanced reading experience
